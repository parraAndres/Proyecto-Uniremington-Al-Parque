# Proyecto-Uniremington-Al-Parque
Uniremington al Parque es una PWA para gestionar impacto social en zonas con baja conectividad. Su arquitectura Offline-First (Angular/Spring Boot) permite registrar beneficiarios y servicios mediante IndexedDB. Digitaliza 9 módulos y 7 facultades para medir la proyección social de forma eficiente y real.
